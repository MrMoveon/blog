# 2、Eslint+Pettier 统一开发规范

先安装vscode的eslint,pettier插件。

为了编辑器和eslint的自动保存格式化一致，需要在项目根目录创建.prettierrc，配置代码格式：统一为使用单引号，无分号等

.prettierrc

{
  "printWidth": 120,
  "trailingComma": "all",
  "singleQuote": true,
  "semi": false,
  "arrowParens": "always"
}
  • printWidth： 控制单行最大长度
  • semi： 控制每条语句是否加上分号
  • tabWidth： 控制一个tab对应的空格数
  • useTabs： 控制缩进用tab还是space
  • singleQuote：控制字符串是用单引号还是双引号
  • trailingComma： 是否去掉末尾的逗号
  • bracketSpacing： 控制json对象括号前后是否加上空格

# 3、vscode配置

# 保存自动格式化

在vscode中打开设置，我们还需要在工作区设置一下自动保存格式化，.prettierrc的配置才会生效。

  1. 搜索format

  2. 选择工作区

  3. 勾选Format On Save，会自动在settings.json中添加

    "editor.formatOnSave": true,// 保存时格式化文件

  1. 打开settings.json，添加如下：

    "editor.codeActionsOnSave": {
    "source.fixAll": true   // 文件保存时，运行自动修复操作
  }

# 窗口失去焦点 保存

有时候会忘记保存代码，我们可以设置当窗口失去焦点时，将自动保存未保存的编辑器:

  1. 搜索save

  2. 选择工作区

  3. 选择Auto Save中的onWindowChange

# 5、 如何用ts在vue3中定义组件

要让 TypeScript 正确推断 Vue 组件选项中的类型，需要使用 defineComponent 全局方法定义组件：

import { defineComponent } from 'vue'

export defaultdefineComponent({
  // 已启用类型推断
})

Vue 对定义了 type 的 prop 执行运行时验证。要将这些类型提供给 TypeScript，我们需要使用 PropType 强制转换构造函数：

<HelloWorld msg="Welcome to Your Vue.js + TypeScript App！！！" :author="{name:'尤玉溪' age: 30 }" />
    



<template>
  <div class="hello">
    <h1>{{ msg }}</h1>
  </div>
</template>    

<script lang="ts">
import { defineComponent, PropType } from 'vue'
// Author接口
interface Author {
  name: string
  age: number
}
export default defineComponent({
  name: 'HelloWorld',
  props: {
    msg: {
      type: String as PropType<string>,
    },
    author: {
      type: Object as PropType<Author>,
      required: true,
      validator(author: Author) {
        return !!author.name
      },
    },
  },
  mounted() {
    console.log(this.author.name, this.author.age)
  },
})
</script>

# 6、如何提取props定义

当我们将props提取出来，发现this.author可能获取不到值，但是我们已经设置了required:true，这是为什么了呢？

因为props是一个Readonly对象，我们需要手动告诉ts，这是Readonly对象，我们在propsType 后面加上as const ,就不会报错了

const propsType = {
  msg: {
    type: String as PropType<string>,
  },
  author: {
    type: Object as PropType<Author>,
    required: true,
    validator(author: Author) {
      return !!author.name
    },
  },
} as const

# 7、 vue的h函数运用

h() 函数是一个用于创建 vnode 的实用程序。 h() 是 createVNode()的简单封装，它接受三个参数：

// @returns {VNode}
h(
  // {String | Object | Function | null} tag
  // 一个 HTML 标签名、一个组件、一个异步组件，或者 null。
  // 使用 null 将会渲染一个注释。
  //
  // 必需的。
  'div',

  // {Object} props
  // 与 attribute、prop 和事件相对应的对象。
  // 我们会在模板中使用。
  //
  // 可选的。
  {},

  // {String | Array | Object} children
  // 子 VNodes, 使用 `h()` 构建,
  // 或使用字符串获取 "文本 Vnode" 或者
  // 有 slot 的对象。
  //
  // 可选的。
  [
    'Some text comes first.',
    h('h1', 'A headline'),
    h(MyComponent, {
      someProp: 'foobar'
    })
  ]
)

在main.ts中实现home组件得内容展示

import { createApp, defineComponent, h } from 'vue'
import router from './router'
import store from './store'
import HelloWorld from './components/HelloWorld.vue'
// 由于ts强类型，这里需要用require，不能用import from,并且当前行禁用eslint
const img = require('./assets/logo.png') // eslint-disable-line
// 定义一个home组件
const Home = defineComponent({
  render() {
    return h('div', { class: 'home' }, [
        h('img', { alt: 'Vue logo', src: img }), 
        h(HelloWorld, { 
            msg: 'Welcome to Your Vue.js + TypeScript App', 
            author: { name: '鱿鱼须', age: 30 } 
        })
    ])
  },
})
//
createApp(Home).use(store).use(router).mount('#app')

# 8、使用jsx开发vue3组件

以 JSX 的方式来编写 Vue 代码 ,需要安装插件 @vue/babel-plugin-jsx

npm install @vue/babel-plugin-jsx -D

配置 Babel

{
  "plugins": ["@vue/babel-plugin-jsx"]
}

在src目录创建App.tsx,并将main.ts中的App.vue组件替换为App

import App from './App'

App.tsx

import { defineComponent, reactive } from 'vue'
import HelloWorld from '@/components/HelloWorld.vue'
const img = require('@/assets/logo.png') // eslint-disable-line

export default defineComponent({
  setup() {
      
    const state = reactive({
      name: '学习vue3+ts',
      desc: '加油加油！',
    })
    
    return () => {
        
      return (
        <div class="home">
          <img alt="Vue logo" src={img} />
          <HelloWorld 
              msg="Welcome to Your Vue.js + TypeScript App！！！" 
              author={{ name: '鱿鱼须', age: 30 }} 
          />
          <p>
            {state.name}-{state.desc}
            <input type="text" v-model={state.name} />
          </p>
        </div>
      )
        
    }
  },
})

# 设置响应式空对象

设置响应式对象 ref , reactive 的时候，如果初始值是空，那么用ref(null)，例如：

const { reactive, ref } = Vue 
// 编辑todo
 let currentTodo = ref(null)
 // 编辑todo
 const editItem = (todo) => {
 	currentTodo.value = todo
 }

注意：如果初始值用reactive，设置值不生效。
