      # ask问答项目前端

      # 基础架构搭建

      # 1、安装脚手架

      npm install -g @vue/cli

      # 2、创建一个项目

      使用命令行进行初始化。

      # 创建项目，选择vue2.0
vue create front

      并配置项目。

      若安装缓慢报错，可尝试用 nrm 切换镜像源进行安装

      # 3、安装ant-design-vue

      npm i --save ant-design-vue

      # 4、按需加载 组件

      babel-plugin-import 是一个用于按需加载组件代码和样式的 babel 插件

      # 安装babel插件
npm install babel-plugin-import --dev

      修改babel.config.js文件，配置 babel-plugin-import

      module.exports = {
  presets: ["@vue/app"],
+  plugins: [
+    [
+      "import",
+      { libraryName: "ant-design-vue", libraryDirectory: "es", style: true }
+    ]
+  ]
};

      由于ant-design-vue依赖于预处理器less，我们还需要安装

      npm install less less-loader -D

      vue.config.js配置less开启javascript

      module.exports = {
  css: {
    loaderOptions: {
      less: {
        lessOptions: {
          javascriptEnabled: true,
        },
      },
    },
  },
}

      最后重启 npm run serve 访问页面，antd 组件的 js 和 css 代码都会按需加载

      # 5、审查项目的 webpack 配置

      审查项目的 webpack 配置,并输出到output.js文件中

      vue inspect > output.js

      # 6、修改网页标题

      // 网页标题
const title = 'M社区'

module.exports = {
  // 对内部的 webpack 配置进行修改
  chainWebpack: (config) => {
    // 修改网页标题
    config.plugin('html').tap((args) => {
      args[0].title = title
      return args
    })
  }
  ...
}

      # 7、修改服务端代理

      // 端口号
const port = 3000
module.exports = {
  ...
  devServer: {
    // 修改端口号
    port,
    // 代理服务
    proxy: {
      '/api': {
        target: 'http://127.0.0.1:7001',
        ws: true,
        changeOrigin: true,
      },
    },
  },
  ...
}

      # 8、scss全局变量注入

      
module.exports = {
  ...
  css: {
    loaderOptions: {
      scss: {
      	//全局变量注入，多个用;分割
        prependData: `@import "~@/assets/style/variables.scss";`,
      },
      less: {
        lessOptions: {
          javascriptEnabled: true,
        },
      },
    },
  },
}

      # 常见问题

      动态路由同页面跳转，报错：NavigationDuplicated: Avoided redundant navigation to current location

      // 报错 NavigationDuplicated: Avoided redundant navigation to current location
this.$router.push({ name: "Detail", params: { id: item.id } });
//解决办法
this.$router.push({ path: `/detail/${item.id}` });
